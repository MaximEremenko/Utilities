Utilities Repository

Welcome to the Utilities repository by Maxim Eremenko.

This project contains client-side tools for RMC6f workflows, diffuse scattering analysis, and visualization.

Live Demos

  • Home Page: Utilities Home
  • Select Subvolume App: RMC6f Select Subvolume App
  • PCA KDE Tool: PCA KDE Tool
    (Kernel Density Estimation + PCA analysis)
  • PCA SDE Tool: PCA SDE Tool
    (Surface Density Estimation + PCA analysis)
  • Diffuse Scattering Tool (Client-side): RMC6f Diffuse Scattering Tool
    (Computes/visualizes/saves diffuse scattering I(h,k,l) in-browser with Type-3 NUFFT CPU/WebGPU backends)
  • Slice Diffuse 3D Tool: RMC Diffuse 3DSlice Tool
  • Background Remover: Background Remover Tool

Repository Structure

  • Utilities/ - Repository root.
  • Utilities/docs/ - GitHub Pages documentation entry.
  • Utilities/RMCProfileUtilities/ - Main tools collection.
  • Utilities/RMCProfileUtilities/Select_Subvolume/ - RMC6f subvolume selector.
  • Utilities/RMCProfileUtilities/PCA_KDE/ - KDE + PCA analysis.
  • Utilities/RMCProfileUtilities/PCA_SDE/ - SDE + PCA analysis.
  • Utilities/RMCProfileUtilities/Diffuse_Scattering/ - Diffuse scattering app.
  • Utilities/RMCProfileUtilities/Diffuse_Scattering/js/ - Modular reader/core/amplitude/scattering-coeff scripts.
  • Utilities/RMCProfileUtilities/RMCDiffuse3DSlice/ - Slice viewer for diffuse 3D DAT files.
  • Utilities/RMCProfileUtilities/Background_Remover/ - Image background utility.

Diffuse Scattering Tool Notes

  • Input: RMC6f configuration files.
  • Output: Diffuse volume and slice views, plus .dat/.json export.
  • Backends: Type-3 NUFFT CPU and WebGPU paths (WebGPU-NUFFT CDN integration).
  • Scattering models: neutron, X-ray, and electron with table/model selection in UI.
